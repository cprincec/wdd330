Notes on DOCUMENT OBJECT MODEL

  • The DOM is a tree-like representation of the contents of a web page.
  • The DOM allows us access and manipulate those contents.
  • getElementByClassName(), querySelectorAll() returns a node-list.
  • A node-list is an array-like object but not an array as it does not have access to all the methods of an array.
  • The Node-list can be converted to an array and then have all array methods act on it.
  • querysSelector() returns the first object that matches the element in parentheses.
  • JQuery is a JavaScript Framework.
  • The classList property is only supported by internet explorer 10. So a function should be used to support older versions.
  • textcontent creates a new text node.
