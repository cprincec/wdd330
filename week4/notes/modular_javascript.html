MODULAR JAVASCRIPT

  • Modules are self contained code comprising of functions, objects, arrays, classes etc that are in a seperate file whose code can be exported for use in another file.
  • For instance a module may contain a class called Human with the class containing properies and methods. This class can be exported using the keyword export and then be imported in a seperate file for use.
  • Modules allow related code to be organized and kept in a seperate file to ensure maintainability, readbility and organization.
