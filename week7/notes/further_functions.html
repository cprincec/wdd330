a function that invokes itself continously until a certain condition is met.

Memoization

storing the result of a function in cache so as to retrieve it if the functions is called again with the same argument instead of executing the function again.

Higher-Order Functions

Higher-order functions are functions that accept another function as an argument, or return another function as a result, or both.
