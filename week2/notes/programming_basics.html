Scripting language

Graceful Degradation

The process of building a website to function well in browsers that supports JavaScript, but still works to a reasonable standard in older browser or if JavaScript or some of its features are unavailable.

Progressive Enhancement

This has to do with building applications that can run with basic functionalities and effects that can run in older browsers or less JavaScript availability, then adding functionalities that can run on updated or modern browsers.

PROGRAMMING BASICS

  • The syntax used by Javascript is known as C-style syntax because its similarities withe the C programming language.
  • Javascript has 7 data types, six of them are primitive
  • Primitive data types include: String, Symbol (introduced in ES6), Number, Boolen, Undefined, Null.
  • Any value that isn't one of the primitive data types listed above is an object.
  • JavaScript objects includes: arrays, functions and object literals.

Variables

  • A variable contains a value stored at memory
  • let and const are used to declare variables
  • variables have to be declared before they can be used.
  • Variables declared with the let and const variables have block scope.
  • Variables declared with var keyword are global scope and can be reassigned.
  • Variables declared outside curly braces are Global scoped
  • Global scope means they can be accessed from anywhere in the program.
  • Variables declared within curly braces (blocks) are local scoped which means they can only be accessed within that block.
  • Global variables should be avoided as much as possible
  • They can cause name clashes in large programs and bring bugs.

Variable Naming

  • Variable and constants names can begin with any of the following:
    lower or uppercase
    dollar sign ($) - discouraged as JQuery uses same nominal rule
    underscore (_) - This is used to idenfy private properties
  • Variables can contain numbers, but cannot start with them.
  • Variable names are case sensitive eg: name is not the same as Name and NAME

Direct Assignment and Assignment by Reference.

  • Assigning a primitive value is know as Direct Assignment
  • When a primitive value is stored in a variable, any changes made are made directly to the value.
    Eg: let a = 1; // a = 1
    let b = a; // b = 1, a = 1
    b = 2; // b = 2, a = 1.
    Notice that the change in made to b did not affect the reference a but rather the value.
  • Assignment by Reference is when a variable stores a non-primitive value.
  • In assignment by reference, changes made will affect all references to that object.
    Eg: let a = {value: 1};
    let b = a; // a.value = 1, b.value = 1
    b.value = 2; // a.value = 2, b.value = 2

Strings

  • JavaScript strings have properties and methods
  • Example of property is the .length which returns the number of characters in the string.
  • Example of method is the toUpperCase() method which converts a lowercase string to uppercase. The charAt() method is use to identify the character at a certain position.
  • The indexOf() method identifies where a certain character appears in a string. If a character does not appear in a string, -1 will be returned

IMPORTANT STRING METHODS

  • toUpperCase(): changes a string to uppercase
  • indexOf(): returns the index of a character in a string.
  • charAt(): returns the character at a specific index in a string
  • lastIndexOf(): returns the index of the last occurence of a character in a string.
  • endsWith(): checks if a string ends with a specific character
  • startsWith(): check if a string starts with a character.
  • includes(): checks if s string contains a character.
  • trim(): removes leading an trailing whitespaces in strings. Including tabs, new line (\n).
  • repeat(): repeats a string a stated amount of time.

Type Coersion

  • A situation where JavaScript tries to convert a value to another data type inorder to perform an operation on it. This isdone automatically/implicitly by Javascript.
  • Automatic type coersion happens in situation like trying to add number of string type (eg "3") to a number of number type, thus:
    "3" * 2 = // will return 6 even though 6 is a string. Javascript coerced the "3" into a number type to enable the multiplication
    Manual type conversion can be done using:
    1. String() - for converting numbers to string.
    2. Number() - for converting numbers to string.
    etc.

Parsing Numbers

  • This is converting numbers represented as string (such as "2", "78") to number. Eg:
    let num = "2"
    parseInt(num) // returns 2
  • Using the parseInt() on decimal numbers of string type returns just the numbers before the decimal point.
    Eg: parseInt("2.9") // returns 2.
  • parseFloat() is used to return a decimal number of string type as a float number data type.
