ARRAYS

  • delete() is used to remove an item at a particular index from an array. for example delete arr[2] // removes item at index 2 and replaces it with undefined.
  • DESTRUCTURING an array is the concept of taking out values in an array and assigning them to variables names in an array such that they can be accessed as individual values.
    Eg: const [x, y] = [1, 2] // in this case, the value of x is 1 and the value of y is 2.

Array Methods:

  • pop(): removes the last item from an array and returns the length of the new array.
  • shift(): removes the first item in an array and returns the new length of the array.
  • push(): adds an item into the end of an array and returns the length of the new array.
  • unshift() : adds an item to the starts of an array and returns the length of the new array.
  • concat(): Creates a new array that contains the items in the old array and the items in array passed into it.
  • The spread operator (...) can also be used to achieve this.
    Eg: let array = ["a", "b"]; let newArray = a.concat(["c", "d"]); // newArray = ["a", "b", "c", "d"]
    or newArray = [...array, ...["c", "d"]]; // this has the same result as the example above.
  • join(): This can be used to turn the items in an array into a single string seperated by whatever you pass in as arguement.
  • slice() : returns an array with items whose index is specified as arguement.
  • splice() : replaces an array item at a particular item with an item / set of items.
  • reverse() : reverses the order of items in an array such that the first becomes last and last becomes first.
  • sort() : sorts an array in alphabetical order. Should not be used for array of numbers.

Set

  • A Set is an object that can store values but the values can never be duplicate.
  • A set can store arrays.
  • Use the new Set() command to create a set.
  • Set methods include: has(), clear(), delete(), add().
  • Sets can be converted to arrays using the spread method.

TENARY OPERATOR

  • This operation has three operands arranged in this format: condition ? code to run if condition is true : code to run if false

For of

  • Used to loop through every item in a set or array.
    example let array = [1, 2, 3]; for (const value of array) { console.log(value)} // this would give 1 2 3 all on a new line.
