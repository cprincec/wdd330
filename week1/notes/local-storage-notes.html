Local Storage

  • Local storage allows data to be stored in a web browser
  • Data stored in local storage has no expiration date but can be removed or deleted
  • Local storage data is stored using the key value pair
  • The localStorage.setItem(key, value) function is used to store data in local storage
  • The localStorage.getItem(key) function is used to retrieve data from the local storage.

The following is a story editor app built with the local storage feature.

In the app, a user can start a story with a title and body then save it.

Stories stored in the app can be retrieved at anytime by entering the title of the story and selecting find it.
